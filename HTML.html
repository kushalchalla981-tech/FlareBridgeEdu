About

FlareBridgeEdu is a lightweight front-end demo which demonstrates connecting educational workflows to the Flare network. This repo contains a simple static front-end: HTML, CSS and JavaScript. It is designed for learning and demo purposes.

Display scholarship opportunities, filter by eligibility and share them with students.
